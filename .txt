Start and End Workflow
🏁 1. Start your session (at home or work)
cd devotional-app   # or wherever your project lives
git stash           # save uncommitted changes just in case
git checkout main   # make sure you're on the right branch
git pull origin main
git stash pop       # recover any local changes

💾 2. Make your changes
Edit files, test, do your thing.

🚀 3. End your session with a clean push
git add .
git commit -m "Update: finished X, started Y"
git push origin main
